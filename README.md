# Java Course Recovery System (APU-CRS)

A robust desktop management system built for academic institutions to automate student course recovery, eligibility validation, and enrolment tracking using Object-Oriented Programming (OOP) principles.

## 🛠️ Technical Stack
* Language: Java (JDK)
* IDE: NetBeans
* Architecture: Advanced implementation of Inheritance, Polymorphism, and Encapsulation to ensure system modularity.
* Data Persistence: Lightweight flat-file storage (CSV/Text) for student profiles and transaction history.

## 🚀 Key Features
* Automated Eligibility Check: Logic-driven module to verify student prerequisites before allowing course entry.
* Secure Enrolment Workflow: End-to-end registration process with real-time data validation.
* Role-Based Access: Dedicated interfaces for Students (Registration/Payments) and Administrators (Record Management).
* Reliable Data Handling: Full CRUD (Create, Read, Update, Delete) capabilities integrated with local text-based databases.
